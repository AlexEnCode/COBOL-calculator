# COBOL Calculator

This project is a calculator developed using the COBOL programming language.

 ------------------------------------------------
|             Calculatrice en COBOL              |
 ------------------------------------------------
Entrez une première valeur, un opérateur puis
une seconde valeur
Liste des operateurs :
- Addition:       +
- Soustraction:   -
- Division:       /
- Multiplication: x
- Puissance:      p
 ------------------------------------------------
Entrez votre calcul:
 
45
+
62
 ------------------------------------------------
 45.00 +  62.00 = 107.00
